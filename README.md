Replace old-string with new-string for all files in path.

### Install

##### Node

```sh
npm i guld-fs-str-replace
```

### Usage

```
await strReplace(path, oldstr, newstr)
```

##### Node

```
const strReplace = require('guld-fs-str-replace')
```
